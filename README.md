# Installation:

* Linux:
  * `sudo apt update`
  * `sudo apt install python3 python3-pip git -y`
  * `git clone https://github.com/i-am-unbekannt/Beacon`
  * `cd Beacon/`
  * `sudo python3 beacon.py`

* Termux:
  * `pkg update`
  * `pkg install python3 python3-pip git -y`
  * `git clone https://github.com/i-am-unbekannt/Beacon`
  * `cd Beacon/`
  * `sudo python3 beacon.py`
